Django REST framework 框架是一个用于构建REST API 的强大而又灵活的工具，DRF框架建立在Django框架基础之上。

<!-- more -->

**DRF特点:**

- 提供了定义序列化器Serializer的方法，可以快速根据 Django ORM 或者其它库自动序列化/反序列化
- 提供了丰富的类视图、Mixin扩展类，简化了视图的编写
- 丰富的定制层级：函数视图、类视图、视图集合到自动生成 API，满足各种需要
- 支持多种身份认证和权限认证方式
- 内置灵活的限流系统
- 直观的 API web 界面
- 具有丰富的插件和强大扩展性
